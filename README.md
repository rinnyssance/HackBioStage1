# HackBio Introductory Assignment

This repository contains my submission for the HackBio introductory assignment.

## Contents
- `hackbio_assignment.R`:  
  An R script that includes:
  - A robust GC% calculator that handles upper and lower case nucleotide sequences
  - A protein molecular weight calculator that returns values in kiloDaltons (kDa)
  - Input validation to return 0 for invalid protein characters

## Notes
This assignment focuses on clarity, reproducibility, and basic biological data handling in R.
